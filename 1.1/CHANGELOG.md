Kv Format 1.1 Specification CHANGELOG
=====================================


[1.1 RC2] - 2026-03-22
----------------------

### Added

- Section on key uniqueness in relation to schemas

- Section on schemas and type annotations


### Changed

- Updated Part I (1.0 normative text) to 1.0 RC4

- Rewrote the Format Declaration section

- Changed syntax for custom types in encoded directives from `!x/b64 <custom-type>` to `!b64 x/<custom-type>`

- Unicode variant formal grammar reframed in terms of rules difference instead of repetition

- Schema grammar formalized, with changed productions: values and no b64-lines


### Deleted

- Introduction before Part I
